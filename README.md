# Question Bank Pro

### 开发规范
#### 接口设计
- 创建表记录： POST/{控制器路由名称}
- 修改表记录： PUT/{控制器路由名称}
- 删除表记录： DELETE/{控制器路由名称}/{id}
- 分页查询记录： GET/{控制器路由名称}/page
- 获取单条记录： GET/{控制器路由名称}/{id}
- 获取列表记录： GET/{控制器路由名称}

#### 服务设计
- 创建：create{控制器路由名称}
- 修改：update{控制器路由名称}
- 删除：delete{控制器路由名称}ById
- 分页查询：query{控制器路由名称}Page
- 获取单条记录：query{控制器路由名称}ById
- 获取列表记录：query{控制器路由名称}List
